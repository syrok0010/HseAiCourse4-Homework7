# Архитектура решения задачи регрессии (Homework 7)

## Общая концепция
Решение построено на базе глубокого обучения (Deep Learning) с использованием библиотеки **PyTorch**. Основной упор сделан на сравнение двух принципиально разных подходов к обработке табличных данных: полносвязных сетей (FCN) и одномерных сверточных сетей (CNN).

## Архитектуры моделей

### 1. FCN (Fully Connected Network)
Реализована как многослойный перцептрон (MLP) с глубокой структурой:
- **Входной слой**: 52 признака.
- **Скрытые слои**: [512, 256, 128, 64].
- **Регуляризация**: 
  - `BatchNorm1d` после каждого линейного слоя для стабилизации активаций.
  - `Dropout` (0.2) для предотвращения переобучения.
- **Функция активации**: `ReLU`.

### 2. CNN (Convolutional Neural Network 1D)
Использует одномерные свертки для поиска паттернов и зависимостей между соседними признаками в векторе:
- **Слоевая структура**: 2 блока свертки (ядро 3, паддинг 1) с BatchNorm.
- **Переход к регрессии**: Сплющивание (Flatten) и финальный полносвязный слой.

## Пайплайн обучения
Процесс управляется через `TrainerOrchestrator`, который реализует следующие шаги:
1. **Препроцессинг**: Загрузка данных из HW4, разделение на train/test.
2. **Масштабирование**: Применение `StandardScaler`.
3. **Логарифмирование**: Перевод таргета (зарплаты) в логарифмическую шкалу (`log1p`).
4. **Обучение**: Использование оптимизатора `Adam` с планировщиком `ReduceLROnPlateau`.
5. **Трекинг**: Автоматическое логирование параметров, метрик и артефактов модели в **MLflow**.

## Связи между заданиями

- **Homework 7** → использование подготовленных данных для обучения нейросетевых регрессоров.
