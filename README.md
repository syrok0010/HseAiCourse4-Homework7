# Homework 7 — Обучение нейронных сетей и трекинг в MLflow

Реализация решения задачи регрессии для датасета HH.ru с использованием глубокого обучения (PyTorch) и системы логирования экспериментов (MLflow).

## Системные требования
- **Операционная система**: Linux (Ubuntu 20.04+) или macOS.
- **Python**: Версия 3.10 или выше.
- **Оперативная память**: Рекомендуется от 4 ГБ.
- **Свободное место**: Минимум 500 МБ для зависимостей и данных MLflow.

## Настройка окружения
Для изоляции зависимостей рекомендуется использовать виртуальное окружение:
1. **Создание окружения**: `python3 -m venv .venv`
2. **Активация**: `source .venv/bin/activate`
3. **Установка зависимостей**: `pip install -r requirements.txt`

## Структура проекта
Проект строго декомпозирован (правило 50 строк и одна сущность на файл):
- `homework7/` — корневой пакет с логикой.
  - `data/` — загрузка, масштабирование и создание батчей.
  - `models/` — архитектуры `FCN`, `CNN` и Sklearn-обертки.
  - `training/` — цикл обучения, метрики и оркестрация.
  - `tracking/` — интеграция с MLflow.
- `docs/` — дополнительная документация.
  - `architecture.md` — подробное описание архитектур и пайплайна.
- `resources/` — сохраненные веса моделей (`.pt`).
- `train.py` — точка входа для запуска обучения.

## Результаты и гиперпараметры
Финальные метрики на тестовой выборке:
- **FCN (R2)**: **0.6074** (Слои: [512, 256, 128, 64], LR: 0.0005)
- **CNN (R2)**: **0.5572** (Слои: Conv1d(32) -> Conv1d(64))

Используемые технологии:
- **Optimizer**: Adam (с Weight Decay `1e-5`).
- **Scheduler**: `ReduceLROnPlateau` (patience=10).
- **Target Transformation**: `log1p` (обучение на логарифме зарплаты).

## Запуск
```bash
export PYTHONPATH=$PYTHONPATH:.
python3 train.py
```
После обучения запустите `mlflow ui` для просмотра интерактивных графиков и сравнения ранов.
